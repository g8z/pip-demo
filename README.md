# pip-demo

Placeholder repository for a **Picture-in-Picture (PiP)** browser / media demo.

Intended to show how video or meeting UI can continue in a floating window while the user navigates the host application. Add your PiP implementation (Media Session API, `documentPictureInPicture`, or platform-specific hooks) when building out the sample.
